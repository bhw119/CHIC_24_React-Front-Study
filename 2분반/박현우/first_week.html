기본 선택자

전체 선택자 *

태그 선택자 태그

아이디 선택자 #아이디

클래스 선택자 .클래스

속성 선택자

선택자[속성]: 특정한 속성이 있는 태그 선택

선택자[속성 = 값]: 특정한 속성 내부 값이 특정 값과 같은 태그 선택

후손 선택자와 자손 선택자

후손 선택자: 특정한 태그의 후손을 선택할 때 사용

선택자A 선택자B: 선택자A의 후손인 선택자B 선택

자손 선택자: 특정한 태그의 자손을 선택할 때 사용

선택자A > 선택자B: 선택자A의 자손인 선택자B 선택

반응 - 상태 - 구조 선택자

반응 선택자: 사용자 반응으로 생성되는 특정한 상태를 선택

사용자가 특정 태그 위에 마우스 커서를 올리면 hover 상태

사용자가 특정 태그를 마우스로 클릭하면 active 상태

상태 선택자: 입력 양식의 상태를 선택할 때 사용

:checked, :focus, :enabled, :disabled

구조 선택자: 특정한 위치에 있는 태그를 선택할 때 사용

first-child : 형제 관계에서 첫 번째로 등장하는 태그 선택

last-child : 형제 관계에서 첫 번째로 등장하는 태그 선택

nth-child : 형제 관계에서 앞에서 수열 번째로 등장하는 태그 선택

nth-last-child : 형제 관계에서 뒤에서 수열 번째로 등장하는 태그 선택

배경 속성: 배경의 이미지나 색상의 지정.

다양한 레이아웃의 구성과 기능

수평,중앙, One True 정렬 레이아웃

수평 정렬 레이아웃: (네이버 사이트의 광고,뉴스스탠드와 로그인,날씨는 수평 정렬 되어 있음)

대부분의 경우: 자손에게 float 속성을 지정하고, 부모의 overflow 속성에는 hidden 키워드를 적용

중앙 정렬 레이아웃: 중앙 정렬하고 싶은 태그에 width 속성을 부여하고 margin 속성을 '0 auto'로 입력

One True 레이아웃: 국내 포털 사이트의 모든 메인 페이지

One True Layout 구성하기: 1. 행 구성하기 2. 열 구성하기 3. 레이아웃 구성하기

요소 배치

절대 위치를 사용한 요소 배치: 자손의 position 속성에 absolute를 적용하려면 부모의 position 속성에 relative를 적용

요소를 중앙에 배치: 1. 중앙 정렬하려는 div 태그의 position 속성을 absolute로 지정 2. left 속성과 top 속성을 모두 50%로 지정 3. 중앙에 정렬하려는 div 태그의 margin-left 속성과 margin-top 속성에 음수 입력(div 태그 너비와 높이의 절반)

요소를 고정 위치에 배치: position 스타일 속성에 fixed를 적용하고 left,top,right,botton 속성에 값을 입력해 위치를 설정.

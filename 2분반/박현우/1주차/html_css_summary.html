웹 사이트의 생성: HTML과 CSS

  • HTML : 어디에 무엇이 있는지 정의, 웹 사이트의 구조
  • CSS : 사이트의 구성요소들이 어떻게 생겼는지
    • ※ HTML : 뼈대 CSS: 그 위에 덧입힘

HTML

  • HEAD : html이라는 언어를 이해할 수 있는 능력(기본 프레임)을 머리에 넣어준 것. 문서 정체의 정보.
  • BODY: 실제 구성요소. 눈으로 확인할 수 있는 부분들

HTML 기본 태그

  • h1 ~ h5 : 제목
  • p : 본문
  • br: 줄바꿈
  • a: 링크
  • li: list
  • ul: unordered list
  • ol: ordered list
  • img: 이미지 삽입
  • input: 사용자로부터 입력을 받음. 다양한 attribute 있음
    input attribute 목록
  • strong : 글자의 강조
  • i: 글자를 이텔릭체로 변경
    • HTML 주요 태그 정리

CSS : Cascading Style Sheets

CSS의 기본구조 : selector{property: value;}

  • Selector: 어떤 요소에 적용할지
  • Property: 어떤 스타일, 어떤 서식을 적용할지
  • value: (변화할)형태
  • /**/: 주석

- HTML에서 CSS 사용하기

  1. style /style - 태그 사용하기
  2. Style = “;” 이용하기
  3. style.css 파일 연동시키기 - 주로 사용

CSS의 Cascading

웹 브라우저 기본 스타일 -> 외부/내부 스타일 시트 -> 인라인 스타일(우선)

selecter의 종류

  • type 선택자
  • attribute 선택자
  • id 선택자
  • class 선택자
  • 그룹 선택자
  • 가상 선택자

대표적인 가상선택자

  1. link : 방문한 적이 없는 링크
  2. visited : 방문학 적이 있는 링크
  3. hover : 마우스를 롤오버 했을 때
  4. active : 마우스를 클릭했을 때
  5. 포커스 되었을 때(input 태그 등)

div와 span

  • Div : block element
  • Span: in-line element - width와 margin 등이 적용되지 않음.

자주 쓰는 tag들의 css 속성(예시)

  • Border-radius:30px;
  • Border-bottom: 0.2px solid sky-blue;
  • Background-color:pink;
  • Background-image:url()

HTML이란?

웹 문서를 만들기 위하여 사용하는 기본적인 웹 언어의 한 종류이다. 하이퍼텍스트를 작성하기 위해 개발되었다.

HTML5 기본 용어 정리

  • 태그와 요소
    • HTML 페이지를 구성하는 각 부품을 요소라고 함
    • 태그는 이러한 요소를 만들 때 사용하는 작성 방법
  • 속성
    • 태그에 추가 정보를 부여할 때 사용
  • 주석
    • 프로그램 실행에 영향을 미치지 않고 설명용으로 사용하는 코드

HTML5 태그

  • head 태그 내부에 넣을 수 있는 태그
    • meta : 웹 페이지에 추가 정보 전달
    • title : 페이지 제목 지정
    • script : 웹 페이지에 스크립트 추가
    • link : 웹 페이지에 다른 파일 추가
    • style : 웹 페이지에 스타일시트 추가
    • base : 웹 페이지의 기본 경로 지정

  • body 태그 내부에 자주 쓰는 태그
    • h1~h6 : 제목 요소
    • p : 단락 나누기
    • br : 줄 바꿈
    • strong : 중요 텍스트 강조
    • a : 하이퍼링크 생성
    • img : 이미지 생성
    • nav : 내비게이션

  • 특수문자 표기(& _ ;)
    • nbsp : 공백
    • lt : <
    • gt : >
    • amp : &

  • 입력 태그
    • text : 글자 입력 양식 생성
    • button : 버튼 생성
    • checkbox : 표의 제목 셀 생성
    • file : 파일 입력 양식 생성

  • 공부하면서 처음 알게된 것
    • table : 표 삽입
    • tr : 표에 행 삽입
    • th : 표의 제목 셀 생성
    • td : 표의 일반 셀 생성

CSS란?

웹 문서의 전반적인 스타일을 미리 저장해 둔 스타일시트이다. 문서 전체의 일관성을 유지할 수 있고, 세세한 스타일 지정의 필요를 줄어들게 하였다.

CSS 선택자 종류

  • 전체 선택자 : *
  • 태그 선택자 : 태그
  • 아이디 선택자 : #아이디
  • 클래스 선택자 : .클래스
  • 후손 선택자 : 선택자 선택자

CSS 주요 태그

  • 박스
    • margin : 요소 외부의 여백 설정
    • border : 요소의 테두리 설정
    • padding : 요소 내부의 여백 설정
    • width : 요소의 너비 설정
    • height: 요소의 높이 설정
  • 배치
    • display : 요소의 박스 형식 설정
    • position : 요소의 위치 설정
    • float : 요소를 좌우로 정렬
    • width : 요소의 너비 설정
    • flex : 플렉스 박스 레이아웃 설정
  • 텍스트
    • color : 텍스트 색상 설정
    • font-size: 텍스트 크기 설정
    • text-align : 텍스트 정렬 설정
    • line-height : 텍스트 줄 간격 설정
  • 배경
    • background-color : 배경 색상 설정
    • background-image : 배경 이미지 설정
    • background-size : 배경 이미지 크기 설정
    • background-position : 배경 이미지 위치 설정
  • 기타
    • opacity: 요소의 투명도 설정
    • border-radius : 테두리 둥글게 만들기
    • font-family : 글꼴 설정
    • text-shadow : 텍스트 그림자 설정
    • overflow : 넘치는 콘텐츠 처리 방법 설정

반응형 웹이란?

웹 페이지 하나로도 데스크톱, 태블릿PC, 스마트폰에 맞게 디자인이 자동으로 반응해서 변경되는 웹 페이지를 의미한다. 반응형 웹 페이지는 미디어 쿼리를 사용해 개발한다.

@media 를 이용하여 쓴다

반응형 웹

  • 유동적인 그리드 레이아웃 : 웹 페이지 요소들이 상대적인 단위(%)로 크기를 지정하여, 화면 크기에 따라 유연하게 조정
  • 미디어 쿼리 : 특정 화면 크기나 장치 유형에 따라 스타일 시트를 조건부로 적용
  • 모바일 최적화 : 모든 장치에서 동일한 콘텐츠에 접근하고, 일관된 인터페이스를 제공하여 사용자의 혼란을 줄임

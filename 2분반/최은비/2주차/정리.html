식별자

식별자는 자바스크립트에서 변수나 함수 등에 이름을 붙일 때 사용하는 단어
사용할 수 있는 예 alpha alpha10 _alpha $alpha AlPha ALPHA
사용할 수 없는 예 break 273alpha has space
설명 키워드를 사용하면 안 됨 특수 문자는 _와 $만 허용 숫자로 시작하면 안 됨 공백은 입력하면 안 됨

alert('Hello World') -> 함수

Array.length -> 속성

input -> 변수

prompt('Message', 'Defstr') -> 함수

Math.PI -> 속성

Math.abs(-273) -> 메서드

사칙연산

+ 덧샘, - 뺼셈, * 곱셈, / 나눗셈

함수

function() 형태로 작성

함수의 괄호 안에 집어넣어 함수 쪽에 추가적인 정보를 전달하는 것을 매개변수라고 함. 함수를 실행한 결과는 반환 값(리턴 값)이라고 함

이러한 함수는 긴 코드를 짧게 줄여서 쓰고 싶을 때 사용함.

비슷한 것을 적용하고 싶을 때 함수를 쓰기도 함

(실습한 것은 html, css 파일에 있음)

문서 객체 선택

1개 선택일 때 : 아이디로 1개 선택 또는 선택자로 1개 선택 (document.getElementById(아이디), document.querySelector(선택자))

여러 개 선택 : name 속성으로 여러개 선택, class 속성으로 여러개 선택, 선택자로 여러개 선택

글자 속성

textContent : 문서 객체 내부 글자를 순수 텍스트 형식으로 가져오도록 변경

innerHTML : 문서 객체 내부 글자의 HTML 태그를 반영해 가져오도록 변경

스타일 식별자 변환(CSS : JS)

background-image : backgroundImage

background-color : backgroundColor

box-sizing : boxSizing

list-style : listStyle

이벤트 모델 종류(이벤트 연결 방식)

DOM 레벨 0 : 인라인 이벤트 모델, 고전 이벤트 모델

DOM 레벨 2 : 마이크로소프트 인터넷 익스플로러 이벤트 모델, 표준 이벤트 모델

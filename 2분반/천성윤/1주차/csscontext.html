CSS - 핵심 CSS 총 정리

✅ Background

  • background-color : 배경색
  • background-image : 배경 이미지 (url("이미지주소"))
  • background-repeat : 배경 이미지 반복 여부
    • no-repeat : 반복하지 않음
    • repeat : 반복함
    • repeat-x, repeat-y : 가로반복, 세로반복
  • background-position : 배경 이미지의 위치 지정 (top, bottom, left, right, center)

✅ Font

1️⃣ font-size

단위 설명
px 고정값, 절대 단위
em 가변 크기(반응형 웹), 상대 단위, 상위 요소의 폰트 사이즈를 상속받음
rem 최상위 요소의 폰트 사이즈를 기준

2️⃣ font-style

속성값 설명
normal 보통 모양
italic 기울임꼴
oblique 기울임꼴

3️⃣ font-weight

속성값 설명
normal 보통 굵기 (숫자 400)
bold 굵은 굵기 (숫자 700)
bolder 상속된 값보다 굵음
lighter 상속된 값보다 얇음
number 100-900

✅ text-align

속성값 설명
left 왼쪽 정렬 (기본값)
right 오른쪽 정렬
center 가운데 정렬
justify 양쪽 정렬

✅ Box model

HTML의 모든 요소는 box로 표현됩니다.

  • content : width, height 속성으로 크기 조절
  • border : 테두리
  • padding : border 내부의 여백
  • margin : border 외부의 여백

1️⃣ box-sizing

속성값 설명
content-box content의 크기만 width, height로 지정
border-box padding, border를 포함하여 크기 지정

✅ Border

1️⃣ border-style

속성값 설명
none 테두리를 표시하지 않음
hidden 테두리를 표시하지 않음
dotted 둥근 점 테두리
dashed 직사각형 테두리
solid 직선 테두리
double 두 개의 평행한 직선 테두리
groove 파인 테두리
ridge 튀어나온 테두리
inset 파인 테두리
outset 튀어나온 테두리

2️⃣ border-width

선의 굵기를 설정하는 속성 (px, pt, cm, em 등).

3️⃣ border-color

모든 면의 테두리 색상을 설정 (예: border-color: red;).

4️⃣ border-radius

모서리를 둥글게 만듦 (예: border-radius: 10px;).

✅ margin

border 외부의 여백 (상하좌우 지정 가능).

✅ padding

content를 둘러싸는 여백 (상하좌우 지정 가능).

✅ Selector

1️⃣ 기본 선택자

  • Type Selector (태그 선택자): HTML 요소 이름 기재
  • Universal Selector (전체 선택자): * 사용
  • Group Selector (그룹 선택자): , 사용하여 여러 요소 선택

2️⃣ id & class Selector

  • ID Selector: #id 사용
  • Class Selector: .class-name 사용

3️⃣ Attribute Selector

  • [attr]: 특정 속성을 가진 요소 선택
  • [attr=value]: 특정 값을 가진 요소 선택
  • [attr~=value]: 단어 포함 요소 선택
  • [attr^=value]: 특정 값으로 시작하는 요소 선택
  • [attr$=value]: 특정 값으로 끝나는 요소 선택
  • [attr*=value]: 특정 값을 포함하는 요소 선택

✅ 결합 선택자

1️⃣ 상속

자손 결합자 (공백)와 자식 결합자 (>) 사용.

2️⃣ 형제

  • 일반 형제 결합자 (~): 같은 레벨의 뒤에 있는 형제 선택
  • 인접 형제 결합자 (+): 바로 뒤에 있는 형제 선택

✅ 가상 클래스 선택자

선택자 설명
a:link 아직 방문한 적이 없는 링크
a:visited 방문한 적이 있는 링크
:hover 마우스를 올렸을 때
:active 마우스를 눌렀을 때
:focus 포커싱 되었을 때
:disabled 비활성화된 input
:enabled 활성화된 input
:checked 선택된 checkbox, radio

✅ 가상 요소 선택자

가상 요소 선택자 설명
::before, ::after 앞이나 뒤에 가상의 요소 추가
::first-letter 첫 번째 글자
::first-line 첫 번째 라인
::selection 선택한 영역

✅ 상속과 우선 순위

1️⃣ 상속

부모 요소의 스타일이 자식 요소에 상속됨.

2️⃣ 우선 순위

선언된 위치, 코드 위치, Specificity(명시도)에 따라 우선순위 결정.

✅ 레이아웃

1️⃣ display

  • inline: 콘텐츠 크기로 영역 설정, width/height 지정 불가
  • block: width/height 지정 가능, 가로줄 전체 차지
  • inline-block: width/height 지정 가능, 가로방향 추가
  • none: 요소 숨기기

2️⃣ float

속성값 설명
left 왼쪽에 뜨도록
right 오른쪽에 뜨도록
none 정렬하지 않음
inherit 부모의 float 값 상속

3️⃣ position

속성값 설명
static 기본값
relative 기준 설정
absolute 문서 흐름에서 제거, relative 기준
sticky 스크롤에 따라 고정
fixed 화면 위치에 고정

4️⃣ z-index

요소가 겹칠 때, 어떤 요소가 제일 위로 올지 설정 (position 필요).

5️⃣ overflow

속성값 설명
visible 넘치면 깨져 보임
hidden 넘치는 콘텐츠 숨김
scroll 스크롤로 볼 수 있음
auto 브라우저가 자동 선택

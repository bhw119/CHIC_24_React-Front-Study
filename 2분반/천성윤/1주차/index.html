HTML 개요

HTML(Hypertext Markup Language)은 웹 페이지를 작성하는 데 사용되는 마크업 언어이다. HTML은 웹 브라우저가 이해할 수 있는 구조를 제공하여 웹 페이지의 내용과 의미를 정의한다.

HTML의 주요 특징

  • 웹 페이지의 구조를 정의하는 마크업 언어
  • HTML 요소를 사용하여 텍스트, 이미지, 링크, 비디오 등을 표현
  • 웹 브라우저가 HTML 코드를 해석하여 웹 페이지를 렌더링
  • HTML5 버전은 웹 표준을 준수하며 다양한 기능 제공

CSS 개요

CSS(Cascading Style Sheets)는 HTML 요소의 스타일을 지정하는 데 사용된다. CSS를 사용하면 웹 페이지의 디자인과 레이아웃을 쉽게 제어할 수 있다.

CSS의 주요 특징

  • HTML 요소의 스타일을 정의하는 언어
  • 색상, 폰트, 크기, 레이아웃 등 다양한 스타일 속성 제공
  • CSS 선택자를 사용하여 특정 HTML 요소를 선택하고 스타일 적용
  • CSS 선언 방식에 따라 인라인, 내부, 외부 스타일시트로 구분
  • CSS 전처리기(SASS, LESS 등)를 사용하여 확장성 있는 스타일 작성 가능

주요 HTML 태그

HTML에는 다양한 태그가 있으며, 각 태그는 특정한 의미와 기능을 가지고 있다.

대표적인 HTML 태그

  • <html>: 문서의 루트 요소
  • <head>: 문서의 메타데이터를 포함하는 부분
  • <body>: 실제로 브라우저에 표시되는 문서의 본문 부분
  • <div>: 구획을 나타내는 블록 요소
  • <p>: 단락을 나타내는 요소
  • <h1>, <h2>, <h3>, <h4>, <h5>, <h6>: 제목 요소
  • <a>: 하이퍼링크를 생성하는 요소
  • <img>: 이미지를 삽입하는 요소
  • <ul>, <ol>, <li>: 목록을 생성하는 요소

주요 CSS 속성

대표적인 CSS 속성

  • color: 글자 색상 설정
  • font-size: 글자 크기 설정
  • font-family: 글꼴 지정
  • background-color: 배경색 설정
  • width: 요소의 너비 설정
  • height: 요소의 높이 설정
  • margin: 요소의 외부 여백 설정
  • padding: 요소의 내부 여백 설정
  • border: 요소의 경계선 설정
  • display: 요소의 표시 방식 설정

HTML과 CSS의 특징

HTML과 CSS는 웹 페이지 개발에 필수적인 기술로, 각각의 특징이 있다.

HTML의 특징

  • 구조를 정의하는 마크업 언어
  • 웹 페이지의 내용과 구조를 나타냄
  • 웹 브라우저가 해석하여 화면에 표시
  • 다양한 요소(태그)를 사용하여 문서 구조 정의

CSS의 특징

  • 스타일을 정의하는 언어
  • HTML 요소의 외관(디자인)을 제어
  • 색상, 글꼴, 배치 등 다양한 스타일 지정 가능
  • CSS 선택자를 사용하여 특정 요소 선택 및 스타일 적용

HTML과 CSS는 웹 페이지 개발에 필수적이며, 서로 밀접하게 연관되어 있다. HTML은 웹 페이지의 구조와 내용을 정의하고, CSS는 이를 효과적으로 스타일링하여 사용자에게 보다 향상된 웹 경험을 제공한다.

주의할 점 및 꿀팁

주의할 점

  • 시맨틱 HTML 사용: 의미 있는 태그를 사용하여 문서 구조를 명확히 표현
  • CSS 선택자 사용 주의: 불필요한 선택자 사용은 성능 저하를 초래할 수 있음
  • CSS 스타일 중복 피하기: 중복된 스타일은 유지보수성을 저하시킬 수 있음
  • CSS specificity 이해하기: 선택자의 우선순위를 잘 파악하여 스타일 적용
  • 반응형 웹 디자인 고려: 다양한 디바이스에 최적화된 레이아웃 구현

꿀팁

  • CSS 프레임워크 활용: Bootstrap, Foundation 등의 프레임워크를 사용하면 빠른 개발 가능
  • CSS 전처리기 사용: SASS, LESS 등의 전처리기를 활용하여 코드 구조화 및 확장성 향상
  • CSS 리팩토링: 코드 정리, 변수 사용, 모듈화 등으로 유지보수성 높이기
  • CSS 린트 도구 사용: CSS 코드 스타일 및 오류 검사를 통해 일관성 유지
  • CSS 문서화: 스타일 가이드 작성, 주석 달기 등으로 팀 협업 및 유지보수 용이

CSS 인터랙션 효과

CSS 인터랙션 기술

  • 호버 효과: :hover 가상 클래스를 사용하여 마우스 오버 시 스타일 변경
  • 애니메이션: @keyframes 규칙과 animation 속성을 사용하여 애니메이션 구현
  • 트랜지션: transition 속성을 사용하여 스타일 변화에 부드러운 효과 적용
  • 변형(Transformation): transform 속성을 사용하여 요소의 크기, 회전, 이동 등 변형

실제 사례

CSS 인터랙션 효과의 예시:

  • 버튼 호버 효과: 버튼에 마우스를 올리면 색상이나 모양이 변경되는 효과
  • 메뉴 드롭다운 애니메이션: 메뉴 항목에 마우스를 올리면 서서히 드롭다운 메뉴가 나타나는 효과
  • 이미지 확대/축소 효과: 이미지에 마우스를 올리면 이미지가 확대되는 효과
  • 스크롤 애니메이션: 페이지를 스크롤할 때 요소가 서서히 나타나는 효과

미디어 쿼리

미디어 쿼리는 CSS를 사용하여 다양한 디바이스와 화면 크기에 맞게 웹 페이지를 최적화할 수 있게 한다.

미디어 쿼리 사용법

  • @media 규칙을 사용하여 미디어 쿼리 정의
  • 미디어 유형(screen, print, speech 등)과 조건(width, height, orientation 등) 지정
  • 지정된 조건에 맞는 경우에만 해당 CSS 스타일이 적용

미디어 쿼리 예시

 /* 화면 너비가 768px 이하인 경우 */ @media (max-width: 768px) { /* 스타일 규칙 */ body { font-size: 14px; } .container { width: 90%; } }
            /* 화면 방향이 가로(landscape)인 경우 /
            @media (orientation: landscape) {
            / 스타일 규칙 */
            .header {
            flex-direction: row;
            }
            }

미디어 쿼리를 활용하면 반응형 웹 디자인을 구현할 수 있다.

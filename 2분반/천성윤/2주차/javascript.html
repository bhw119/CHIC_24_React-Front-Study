자바스크립트와 문서 객체 모델(DOM) 기본

자바스크립트 기본 문법

기본 용어와 출력 방법

자바스크립트는 웹 페이지의 동적 기능을 구현하기 위한 프로그래밍 언어이다. 웹 브라우저에서 실행되며, HTML 문서와 함께 사용된다.

콘솔에 출력을 하려면 console.log()를 사용한다. 이는 개발자가 디버깅을 위해 자주 사용하는 방법이다.

console.log("Hello, JavaScript!");

자료형과 변수형

자바스크립트의 기본 자료형에는 다음과 같은 것들이 있다:

  • 숫자(Number): 정수와 실수를 포함한다. 예: let age = 25;
  • 문자열(String): 텍스트 데이터를 포함한다. 예: let name = "Alice";
  • 불리언(Boolean): 참(true)과 거짓(false) 값을 가진다. 예: let isStudent = true;
  • null: 비어 있음을 나타냅니다. 예: let emptyValue = null;
  • undefined: 값이 할당되지 않음을 나타낸다. 예: let notDefined;
  • 객체(Object): 키-값 쌍의 모음이다. 예: let person = { name: "John", age: 30 };

변수는 var, let, const 키워드를 사용하여 선언한다. let과 const는 블록 범위 변수를 선언하는 반면, var는 함수 범위 변수를 선언한다. 


// 변수 선언
var oldWay = "이전 방식"; // var
let modernWay = "최신 방식"; // let
const constantValue = 3.14; // const (변경 불가)

조건문과 반복문

조건문은 특정 조건에 따라 코드 블록을 실행한다. if, else if, else 문을 사용한다.


// 조건문 예제
let age = 20;

if (age < 18) {
    console.log("미성년자이다.");
} else if (age >= 18 && age < 65) {
    console.log("성인이다.");
} else {
    console.log("노년이다.");
}

반복문은 특정 코드 블록을 여러 번 실행한다. for, while 문을 사용한다.


// for 반복문 예제
for (let i = 0; i < 5; i++) {
    console.log("숫자: " + i);
}

// while 반복문 예제
let count = 0;
while (count < 5) {
    console.log("카운트: " + count);
    count++;
}

함수

함수는 특정 작업을 수행하는 코드 블록이다. function 키워드를 사용하여 정의한다.


// 함수 정의
function greet(name) {
    return "Hello, " + name + "!";
}

// 함수 호출
console.log(greet("Alice"));

객체

객체는 속성(property)과 메서드(method)를 포함하는 데이터 구조이다. 객체는 중괄호 {}를 사용하여 정의한다.


// 객체 정의
let person = {
    name: "John",
    age: 30,
    greet: function() {
        console.log("Hello, my name is " + this.name);
    }
};

// 객체의 메서드 호출
person.greet();

문서 객체 모델(DOM)

문서 객체 모델의 기본 용어와 개념

DOM(Document Object Model)은 HTML 문서를 트리 구조로 표현한 것이다. 각 요소는 객체로 표현되며, 자바스크립트를 사용하여 이러한 요소들을 조작할 수 있다.

문서 객체 선택

DOM 요소를 선택하기 위해 다양한 메서드를 사용할 수 있다:

  • document.getElementById: ID를 사용하여 요소를 선택한다.
  • document.querySelector: CSS 선택자를 사용하여 첫 번째 요소를 선택한다.
  • document.querySelectorAll: CSS 선택자를 사용하여 모든 요소를 선택한다.

// 요소 선택
let elementById = document.getElementById("myElement");
let elementByClass = document.querySelector(".myClass");
let elementsByClass = document.querySelectorAll(".myClass");

문서 객체 조작

선택한 DOM 요소의 내용이나 스타일을 변경할 수 있다.


let element = document.getElementById("myElement");
element.textContent = "새로운 내용"; // 요소의 텍스트 내용 변경
element.style.color = "red"; // 요소의 스타일 변경

이벤트

이벤트는 사용자나 브라우저가 수행할 수 있는 동작이다. 이벤트를 처리하기 위해 이벤트 리스너를 사용할 수 있다. 예를 들어, 클릭 이벤트를 처리할 수 있다.


let button = document.getElementById("myButton");
button.addEventListener("click", function() {
    alert("버튼이 클릭되었다!");
});

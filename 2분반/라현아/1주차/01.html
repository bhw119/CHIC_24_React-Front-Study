웹 프로그래밍 정리

웹 표준 및 관련 기술

  • W3C: HTML 표준을 비롯한 웹 표준안을 제작하거나 제안하는 일을 하는 국제적인 웹 표준화 단체
  • Plugin: 사용자 PC에 프로그램을 추가로 설치해 웹 프라우저의 기능을 확장하는 방법
  • Web 2.0: 서로 다른 사용자가 함께 새로운 콘텐츠를 창조할 수 있는 시대
  • WHATWG: Web Application 1.0 표준 -> HTML5
  • ECMAScript: 자바스크립트의 표준 명칭
  • 일렉트론: HTML5 기반의 데스크톱 애플리케이션 개발 엔진

HTML5 주요 기능

  • 멀티미디어 기능
  • 그래픽 기능

  • 오프라인 및 저장소
  • 시맨틱

  • 웹의 성능 극대화 및 통합

HTML 표기법

  • HTML 표기법: <img> <br> <hr>
  • XHTML 표기법: <img /> <br /> <hr /> => 내용을 가질 수 없는 태그라는 것을 표기

텍스트 스타일

  • <small>작은 글자</small>
  • <sub>아래 첨자</sub>
  • <sup>위 첨자</sup>
  • <ins>밑줄 글자</ins>
  • <del>취소선이 그어진 글자</del>

기타 개념

  • Vendor prefix: 웹 브라우저 공급 업체에서 제공하는 실험적인 기능을 사용
  • 레이아웃
    • 수평 정렬: 부모(overflow: hidden), 자식(float)
    • 중앙 정렬: width, margin: 0 auto
    • One True: "행을 독립적으로 생각"
  • 글자 생략: ellipsis 클래스
  • Media query: 미디어 타입에 따라 CSS를 적용할 수 있는 기능 ~> 반응형 웹 페이지
  • 반응형 웹
    • 뷰포트 설정: name 속성에 viewport가 입력된 meta 태그 = viewport meta 태그
    • 미디어 쿼리 설정: @-규칙@-rule = 스타일시트 내부에서 특정한 규칙을 표현하는 데 사용
